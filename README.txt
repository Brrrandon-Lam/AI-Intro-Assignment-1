TO COMPILE:

1) Open terminal and enter the following command:

g++ main.c -o search_algos

2) Run any of the algorithms in the following format.

./search_algos initial_state_file.txt goal_state_file.txt mode output_file.txt

Modes available are: "iddfs", "dfs", "bfs", "astar"
